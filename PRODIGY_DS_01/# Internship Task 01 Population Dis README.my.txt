# Internship Task 01: Population Distribution Visualization

## 📋 Task Description
The objective of this task was to create a data visualization that represents the population distribution of India by age groups for the year 2022. This task involves translating raw demographic data into a clear, informative visual format (Bar Chart) that highlights the demographic dividend of the population.

## 📊 Visualization Highlights
- **Data Source:** UN World Population Prospects 2022.
- **Visualization Type:** Horizontal Bar Chart (chosen for readability of categorical age groups).
- **Key Features:**
  - Distinct color coding for different age segments (Youth, Working Age, Elderly).
  - Explicit labeling of percentage distribution and absolute population (in Millions).
  - Clear annotation of total population and median age.
